Linux基础

  1. Linux版本
  2. 帮助命令

Linux版本
  • 内核版本
    • 主版本号，次版本号，末版本号
    • 次版本号奇数是开发版，偶数是稳定版
  • 发行版本
    • RedHat Enterprise Linux
    • Fedora
    • CentOS
    • Debian
    • Ubuntu
帮助命令
  • man
    • manual 一共9章 可以通过 man 7 man 或者man man查看每个章节类型是什么
    • man 1 ls 一般情况下 1是省略的
    • man -a passwd 可以查看不同章节重复的名利
  • help
    • shell 命令解释器自带的称为内部命令 其他是外部命令
    • 内部命令 help cd
    • 外部命令 ls —help
    • 可以用 type cd 来看是什么类型命令
  • info
    • info 帮助比 help 更详细，作为 help 的补充
    • info ls
常用命令
  • ls
    • / 是 根目录 /root 是 root 用户的家目录
    • /opt 是绝对路径 ./opt 相对路径 ../opt 也是相对路径 opt也是相对路径
    • 可以追加文件 ls / /root (/ 和 /root 之间一定要有空格多个无所谓
    • ls -l
      • 第一个参数 文件类型和权限 第一个字符是类型 之后九个是权限
      • 第二个参数 文件个数或者文件夹内的个数
      • 第三个参数 是谁创建这个文件
      • 第四个参数 是创建者属于那个用户组
      • 第五个参数 是文件大小
      • 第六个参数 是最后修改的时间
      • 第七个参数 是名字
    • ls -a
      • 可以查看隐藏文件
      • 隐藏文件以.开头就是隐藏文件
    • ls -r 逆序
    • ls -t 时间
    • ls -l -r -t == ls -lrt == ls -ltr 顺序无关
    • ls -R 递归查看下面的全部文件
  • pwd
    • 查看路径
  • clear
    • 清空终端
    • 快捷键 control + l
  • cd
    • shell 提供的补全功能 Tab键
    • cd - 返回上一次呆的路径
