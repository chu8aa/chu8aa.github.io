拓展欧几里得算法
边界情况

当$x a + y b = gcd(a,b) = a $ 此时 $b = 0 , gcd(a,b) = a$

所以$x = 1 , y = 0 ，x = 1$是一定 $y=0$ 是规定

一般情况

$x_1 a + y_1 b = x_2 b + y_2 (a \mod b)$

$ a \mod b = a - \lfloor {\frac ab} \rfloor $

$ x_1 a + y_1 b = x_2 b + y_2 (a - \lfloor{\frac a b}\rfloor) b = y_2 a + (x_2 -\lfloor{\frac ab}\rfloor y_2) b $

$x_2,y_2$是上一层递归求的 $x_1,y_1$是这一层要求的

$x_2 = y ,y_2 = x$
